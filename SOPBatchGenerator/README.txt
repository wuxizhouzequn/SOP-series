SOP Batch Generator

Install location:
%LOCALAPPDATA%\SOPBatchGenerator

How to use:
1. Open SOP Batch Generator from the desktop shortcut.
2. Select the SOP input root folder.
3. Select the output folder.
4. Upload or select logo, marker image, and fonts.
5. Adjust preview layout and layer positions.
6. Click batch export.

Folder rule:
The first folder level under the SOP root is used as the large title.
The second folder level is used as the medium title.
Videos and images are both exported as tasks.

Note:
This installer includes ffmpeg.exe. Users do not need to install FFmpeg separately.
Assets and history are saved under the current Windows user's AppData\Roaming\sop_renderer folder.
